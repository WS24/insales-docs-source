# setup.json

Дефолтный контент блоков виджета

## Шаблон блока

Указывается в файле info.json

`"block_template_handle": "system-benefit-2",`

## Привязка файла

В качестве файла указывается id файла из аккаунта `http://file-store.myinsales.ru/`
